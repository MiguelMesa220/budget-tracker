# budget-tracker
full stack, web-based, budget tracking application.
visualizes spending habits and tracks transactions.

# stack
- React for frontend
- Node.js and express for backend
- SQL database
- Prisma ORM

# features
- account login/registration with authentication
- transaction tracking (add, remove, modify)
- category tracking
- monthly summary
- search/filtering

# structure
- /client (frontend)
- /server (backend)
- /docs (planning)
