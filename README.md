# budget-tracker
full stack, web-based, budget tracking application.
visualizes spending habits and tracks transactions.

# stack
- Frontend: React
- Backend: Node and Express
- Database: PostgreSQL
- DB access: node-postgres (pg)

# features
- account login/registration with authentication
- transaction tracking (add, remove, modify)
- category tracking
- monthly summary
- search/filtering

# structure
- /client (frontend)
- /server (backend)
- /docs (planning)
